# Exercise 1: Semantic Markup

**Semantic elements:**
1. `<header>` - Top section of the page with heading and navigation.
2. `<nav>` - Navigation links list with `aria-label`.
3. `<main>` - Primary content area of the page.
4. `<footer>` - Bottom section with supplementary information.
5. `<time>` - Machine-readable date/time with `datetime` attribute.

**Content sectioning elements:**
1. `<article>` - Single recipe block with title, description and sections.
2. `<section>` - Recipe section grouping - ingredients/instructions.
3. `<aside>` - Supplementary content tangential to its parent section.

**Text and heading elements:**
1. `<h1>` - Page title.
2. `<h2>` - Recipe title.
3. `<h3>` - Section titles.
4. `<p>` - Text content, paragraphs.

**List elements:**
1. `<ul>` - Unordered list. Ingredients list.
2. `<ol>` - Ordered list. Step by step instructions.
3. `<li>` - Individual list item.

**Other standard elements:**
`<html>`, `<head>`, `<body>`, `<meta>`, `<title>`, `<style>`, `<a>`

**Head elements:**
1. `<meta name="description">` - SEO page description.
2. `<meta name="keywords">` - SEO keywords.
3. `<meta name="author">` - Page author metadata.
4. `<link rel="icon">` - SVG favicon for the browser tab.
